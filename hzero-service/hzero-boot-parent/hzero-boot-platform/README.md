## Introduction

平台基础客户端，封装了许多常用的API信息到客户端中，便于在程序中进行使用。例如值集客户端、编码规则客户端、配置维护客户端、数据源客户端等等。

## Dependencies

* 服务 maven 坐标

```xml

    <artifactId>hzero-boot-platform</artifactId>
    <version>${hzero.boot.version}</version>
</dependency>
```

* 依赖服务
    - hzero-platform
